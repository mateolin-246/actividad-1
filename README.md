                                  *EJERCICIO 1*
Gestión de estudiantes con materias y notas. Calcula promedio, identifica la mejor y peor materia, y determina si el estudiante aprueba. Valida que haya mínimo 3 materias y notas entre 0.0 y 5.0.
                                  *EJERCICIO 2*
Recibe un valor por teclado y aplica un impuesto del 19% y un descuento del 10% si el valor base supera los $200.000. Muestra el desglose y el total final.
                                  *EJERCICIO 3*
Genera una serie Fibonacci desde una posición de inicio y por un número de iteraciones definidos por el usuario. Muestra la lista completa, la cantidad de términos y el último valor.
                                  *EJERCICIO 4*
interactivo con tres opciones: verificar si un número es primo, calcular el factorial de un número, o contar las vocales de un texto con conteo individual por vocal.
                                  *EJERCICIO 5*
Solicita nombre, edad y correo con validaciones: nombre mínimo 3 caracteres, edad entre 0 y 120, correo con @ y dominio .com o edu.co. Almacena los registros en una lista y permite consultarlos.
